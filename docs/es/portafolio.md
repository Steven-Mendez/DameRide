# Portafolio

DameRide forma parte de mi portafolio profesional. El proyecto muestra cómo una idea funcional puede transformarse en una aplicación móvil con backend real, arquitectura clara y documentación reproducible.

## Aportes del Proyecto

- Desarrollo móvil con Expo y React Native.
- Arquitectura con Expo Router y TypeScript estricto.
- Integración full stack con Supabase.
- Autenticación por email y Google OAuth.
- Base de datos relacional con RLS, RPCs, triggers e índices.
- Uso de PostGIS para búsquedas geoespaciales.
- Storage para imágenes de perfil y vehículos.
- Realtime para actualizar viajes y reservas.
- Documentación técnica en español.

## Contexto de Autoría

Br. Vilma Paiz aportó la idea de negocio, diseño conceptual, flujos principales y guía funcional. Desarrollé este proyecto para apoyarla en su clase de Emprendimiento en la UAM Managua, y construí enteramente la app, su integración técnica y su documentación.

## Valor Profesional

El repositorio demuestra capacidad para trabajar de punta a punta: interpretar una propuesta funcional, diseñar arquitectura, construir UI móvil, integrar backend, asegurar datos con RLS, documentar decisiones y preparar una base reconstruible.
