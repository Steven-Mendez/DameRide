# Introducción

DameRide es una aplicación móvil para compartir viajes. Permite buscar rutas disponibles, publicar viajes como conductor, reservar asientos, gestionar perfil y administrar vehículos.

El proyecto está construido con Expo, React Native, TypeScript y Supabase. Su backend incluye autenticación, base de datos PostgreSQL con PostGIS, almacenamiento de imágenes, políticas RLS, RPCs de reserva y consultas geoespaciales.

## Rol de Desarrollo

Codifiqué y desarrollé enteramente la app. Esto incluye estructura móvil, pantallas, navegación, integración con Supabase, autenticación, base de datos, almacenamiento, mapas, documentación y scripts de validación.

## Valor Como Portafolio

El proyecto demuestra habilidades full stack aplicadas a un producto móvil: arquitectura con Expo Router, UI en React Native, integración con backend, seguridad básica con RLS, manejo de archivos, tipos generados, realtime y documentación técnica reproducible.

## Alcance Actual

El repositorio contiene flujos funcionales para autenticación, onboarding, búsqueda, publicación, reserva, perfil y vehículos.
