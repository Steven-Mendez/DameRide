# Portfolio

DameRide is part of my professional portfolio. The project shows how a functional idea can become a mobile application with a real backend, clear architecture, and reproducible documentation.

## Project Contributions

- Mobile development with Expo and React Native.
- Architecture with Expo Router and strict TypeScript.
- Full stack integration with Supabase.
- Email and Google OAuth authentication.
- Relational database with RLS, RPCs, triggers, and indexes.
- PostGIS for geospatial searches.
- Storage for profile and vehicle images.
- Realtime updates for rides and reservations.
- Technical documentation in English and Spanish.

## Authorship Context

Br. Vilma Paiz contributed the business idea, conceptual design, main flows, and functional guidance. I developed this project to support her Entrepreneurship class at UAM Managua, and I built the full app, technical integration, and documentation.

## Professional Value

The repository demonstrates end-to-end capability: interpreting a functional proposal, designing architecture, building mobile UI, integrating a backend, securing data with RLS, documenting decisions, and preparing a rebuildable backend foundation.
