# Introduction

DameRide is a mobile app for shared rides. It allows users to find available routes, publish rides as drivers, reserve seats, manage their profile, and manage vehicles.

The project is built with Expo, React Native, TypeScript, and Supabase. Its backend includes authentication, a PostgreSQL database with PostGIS, image storage, RLS policies, reservation RPCs, and geospatial queries.

## Development Role

I coded and developed the entire app. This includes the mobile structure, screens, navigation, Supabase integration, authentication, database, storage, maps, documentation, and validation scripts.

## Portfolio Value

The project demonstrates full stack skills applied to a mobile product: architecture with Expo Router, React Native UI, backend integration, basic security with RLS, file handling, generated types, realtime updates, and reproducible technical documentation.

## Current Scope

The repository contains functional flows for authentication, onboarding, search, publishing, reservations, profile, and vehicles.
