DameRide targets Nicaraguan university students.

**Why:** Domain decisions (departments, phone format +505, university list, carnet/student ID) and copy choices (Spanish, no English fallback in UI) flow from this.

**How to apply:** All user-facing strings should be in Spanish (matching existing onboarding tone). When adding domain data (universities, departments), use Nicaraguan references. The app is currently in demo state — heavy validation is not required for new student-facing fields unless explicitly asked.
